<h2>Amsterdam property analysis</h2>
<p>Moved to live in Amsterdam and decided to make a local property data analysis</p>
<p>
    Data sources:<br>
    <ul>
        <li>Wiki</li>
        <li>Amsterdam.nl</li>
        <li>Foursquare API</li>
    </ul>
</p>
<p>
    Libraries used in this project:<br>
    <ul>
        <li>requests</li>
        <li>numpy</li>
        <li>pandas</li>
        <li>geopandas</li>
        <li>scipy</li>
        <li>geopy</li>
        <li>googletrans</li>
        <li>bs4</li>
        <li>matplotlib</li>
        <li>folium</li>
        <li>seaborn</li>
        <li>sklearn</li>
    </ul>
</p>
